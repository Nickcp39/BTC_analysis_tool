# 减半 / 峰值日期数据

**数据文件**：`data/btc_halving_peak_dates.xlsx`

用于记录各轮 **减半日（Halving）** 与 **当轮大顶日（Peak）**，供 step02、时间模型、逆向推算等统一读取。

| 列名 | 说明 |
|------|------|
| cycle | 周期标签（按峰值年：2013/2017/2021/2025） |
| halving_date | 该轮减半日期 |
| peak_date | 该轮大顶日期（观测或修正值） |

- **2025 轮**：减半 2024-04-20；峰值按本次观测记为 **2025-10-04**，可在 Excel 中随时修改。
- 代码中通过 `code/` 下的加载函数读取此表，避免多处写死日期。
